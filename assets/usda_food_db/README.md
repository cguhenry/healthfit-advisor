USDA FoodData Central data should be downloaded from the official USDA FDC download page:
https://fdc.nal.usda.gov/download-datasets

Recommended dataset for this project:
Foundation Foods CSV, latest release.

Expected extracted files:
- food.csv
- food_nutrient.csv
- nutrient.csv

Old AI-generated files are not used because they do not contain complete nutrient values.
